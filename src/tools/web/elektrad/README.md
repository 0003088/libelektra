# elektrad


## Installation

 * first, install [libelektra](http://libelektra.org/)
 * now run `npm install`


## Running

```
npm start
```

to run in verbose mode (full logs):

```
npm run start:verbose
```

to run in development mode (full logs & running from source):

```
npm run start:dev
```


## Accessing the API

elektrad runs on http://localhost:1234
